# Code of Conduct

Query Vitals aims to be a practical, respectful open-source project for developers working on database performance.

## Expected behavior

- Be respectful and constructive.
- Assume good intent, but accept correction.
- Keep feedback specific and actionable.
- Welcome newcomers and explain project context when it helps.

## Unacceptable behavior

- Harassment, discrimination, or personal attacks.
- Publishing someone else's private information.
- Repeated disruptive comments that derail technical discussion.
- Dismissing security or privacy concerns without investigation.

## Enforcement

Maintainers may edit, hide, or remove comments and issues that violate this code of conduct. Repeated or severe violations may result in a temporary or permanent ban from project spaces.
